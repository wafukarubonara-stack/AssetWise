# AssetWise

Personal asset tracker for Android - Bank, Investment & Crypto management

個人資産トラッカー - 銀行・投資・暗号資産を一元管理

## Features (Planned)
- Multi-account management (Bank, Brokerage, Crypto, Cash)
- Real-time valuation via API (Exchange Rate, Stock, Crypto)
- Dashboard with charts and asset allocation
- Biometric authentication & encrypted database
- Multi-language support (English / Japanese)

## Tech Stack
- **Language:** Java
- **UI:** Material Design 3 + XML Layouts
- **Database:** Room (SQLite) with complex relations
- **Network:** Retrofit2 + OkHttp + Gson
- **Architecture:** MVVM + Repository Pattern + Clean Architecture
- **Security:** BiometricPrompt + EncryptedSharedPreferences
- **Async:** LiveData + Executor

## Development Environment
- Android Studio on Ubuntu 22.04
- Target: API 26+ (Android 8.0+)
- Test device: Google Pixel 9 (Android 15)
